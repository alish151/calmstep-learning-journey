# Calm Step

Calm Step is an educational web application designed to help children with autism improve their learning, plan tasks, and access useful study materials across different school subjects.

## Project Overview
The project allows users to:  
- Browse educational materials online  
- Generate and solve problems in sections like Mathematics, Logic, Reading, and more  
- Track progress and completed tasks  
- Use a clean and modern interface for easier learning

Calm Step is built using a modern frontend stack to ensure a fast and interactive experience.

## Technologies Used
- **React** – for building a dynamic user interface  
- **TypeScript** – for type-safe and reliable code  
- **Vite** – for fast development and build  
- **Tailwind CSS** – for styling and responsive design  
- **shadcn-ui** – for ready-to-use UI components

## How to Run the Project Locally
1. Install Node.js and npm:  
   Download from [https://nodejs.org/](https://nodejs.org/) and check installation:  
   ```bash
   node -v
   npm -v
2. Clone the repository:  
```bash
git clone https://github.com/alish151/calmstep-learning-journey.git
cd calmstep-learning-journey
3. Install dependencies:
```bash
npm install
4. Start the development server:
```bash
npm run dev
5. Open in browser:
Go to http://localhost:5173 to see the project.
